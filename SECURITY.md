# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, **please do NOT open a public issue**.

Instead, report it privately via GitHub's [Security Advisories](https://github.com/lphenom/core/security/advisories/new) feature,
or email the maintainers directly (see the GitHub profile for contact information).

We will acknowledge the report within **72 hours** and aim to release a patch within **14 days** for confirmed vulnerabilities.

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)
